##  GIT changes

```bash
# status
$ git status 

# fetch changes
$ git fetch 

# pull changes
$ git pull

# push changes
$ git push

## Reset hard
$ git reset --hard

## Reset soft
$ git reset --soft 

## Git UI
$ gitk

# Add all files
$ git add .

# Commit changes
$ git commit -m "message"
```
